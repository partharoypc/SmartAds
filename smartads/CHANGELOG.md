# Changelog

All notable changes to the **SmartAds** library will be documented in this file.

## [3.0.0] - 2025-12-23

### Added
- **Frequency Capping**: Limit the frequency of full-screen ads (Interstitial/Rewarded) to improve user experience.
- **Ad Inspector**: Integrated Google Ad Inspector for easier on-device ad troubleshooting.
- **Privacy Options (UMP)**: Full support for the Google User Messaging Platform (UMP) to handle GDPR and CCPA consent.
- **Collapsible Banner Support**: Easily enable specialized collapsible banners via configuration.
- **Native Ad Templates**: Improved support for Small, Medium, and Large native ad layouts.
- **Android 15 Support**: Targeted API Level 35 and updated core dependencies.

### Changed
- **AdMob SDK Update**: Upgraded to Google Mobile Ads SDK `v24.9.0`.
- **Streamlined Configuration**: Centralized all settings within `SmartAdsConfig`.
- **Improved Lifecycle Safety**: Enhanced memory management and lifecycle awareness for all ad formats.

### Removed
- **Revenue Tracking (ILRD)**: Removed `onAdRevenuePaid` callbacks and associated logic to simplify the library and reduce external dependencies.
