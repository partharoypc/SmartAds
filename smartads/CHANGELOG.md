# Changelog

All notable changes to the **SmartAds** library will be documented in this file.

## [5.5.0] - 2026-02-06

### Added
- **Loading Dialog Customization**: Added methods in `SmartAdsConfig` to customize the loading dialog's text, background color, text color, and progress bar color at runtime.
- **Theme Support**: Introduced `Theme.SmartAds.Fullscreen` to ensure proper rendering of Material Components in internal activities.

### Fixed
- **House Interstitial Crash**: Resolved `InflateException` in `HouseInterstitialActivity` by migrating to `AppCompatActivity` and applying the correct Material theme.
- **UI Contrast**: Improved `LoadingAdDialog` default background color (now White) for better text readability.
- **Connectivity Checks**: Enhanced `NetworkUtils` to reliably detect internet connection across different Android API levels.

### Changed
- Refined `SmartAdsConfig` builder pattern for better chaining of UI customization options.
- **House Ads System**: Built-in support for internal cross-promotion with weighted random selection and rich UI (Icon, Media, Rating, CTA).
- **Comprehensive Mediation Verification**: Added `verifyMediation()` tool to detect Meta, AppLovin, and Unity adapters at runtime.
- **Adaptive & Collapsible Banners**: Native support for building modern, high-conversion banner layouts.
- **Enhanced Native Templates**: Full support for Small, Medium, and Large native templates alongside completely custom XML layouts.
- **AdMob Inspector & Test Suite**: Easy-to-use hooks for launching official AdMob debug and mediation test tools.
- **Multi-SDK Calibration**: Targeted API Level 36 and updated all core dependencies for maximum performance.

### Changed
- **AdMob SDK Update**: Upgraded to Google Mobile Ads SDK `v24.9.0`.
- **Streamlined Configuration**: Centralized all settings within `SmartAdsConfig.Builder` with over 20+ customizable parameters.
- **Premium UI**: Redesigned the full-screen ad loading dialog with customizable colors and animations.
- **Lifecycle Excellence**: Optimized memory management for all ad formats to ensure zero Activity-leaks.

## [3.0.0] - 2025-12-23

### Added
- **Frequency Capping**: Limit the frequency of full-screen ads (Interstitial/Rewarded) to improve user experience.
- **Ad Inspector**: Integrated Google Ad Inspector for easier on-device ad troubleshooting.
- **Privacy Options (UMP)**: Full support for the Google User Messaging Platform (UMP) to handle GDPR and CCPA consent.
- **Collapsible Banner Support**: Easily enable specialized collapsible banners via configuration.
- **Native Ad Templates**: Improved support for Small, Medium, and Large native ad layouts.
- **Android 15 Support**: Targeted API Level 35 and updated core dependencies.

### Changed
- **AdMob SDK Update**: Upgraded to Google Mobile Ads SDK `v24.9.0`.
- **Streamlined Configuration**: Centralized all settings within `SmartAdsConfig`.
- **Improved Lifecycle Safety**: Enhanced memory management and lifecycle awareness for all ad formats.

### Removed
- **Revenue Tracking (ILRD)**: Removed `onAdRevenuePaid` callbacks and associated logic to simplify the library and reduce external dependencies.
